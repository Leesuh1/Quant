# 📈 AI 기반 투자 포트폴리오 분석

재무 데이터를 기반으로 기업의 성장성을 분석하고,
데이터 기반 투자 전략을 통해 포트폴리오를 구성하는 데이터 분석 프로젝트입니다.

본 프로젝트에서는 **기업 재무 데이터 → 성장 지표 계산 → 종목 랭킹 → 포트폴리오 구성 → 성과 분석**의 과정을 수행했습니다.

---

# 📌 프로젝트 목적

기업의 재무 데이터를 기반으로 성장성이 높은 기업을 선별하고
이를 활용한 투자 전략이 **시장 대비 초과 수익을 낼 수 있는지 분석**하는 것을 목표로 합니다.

특히 다음과 같은 질문을 해결하고자 합니다.

* 재무 데이터 기반 성장률 지표는 투자 전략으로 활용할 수 있는가?
* 성장성이 높은 기업을 선별하면 시장 대비 초과 수익을 낼 수 있는가?
* 단순한 점수 기반 랭킹 전략은 실제 포트폴리오에서 어떤 성과를 보이는가?

---

# 📊 분석 프로세스

본 프로젝트는 다음 단계로 진행되었습니다.

```
데이터 수집
   ↓
데이터 전처리
   ↓
재무 성장률, 수익성 지표 계산
   ↓
투자 점수(Scoring) 계산
   ↓
종목 랭킹 및 포트폴리오 구성
```

---

# 📂 데이터

본 프로젝트에서는 기업 재무 데이터를 기반으로 분석을 수행했습니다.

주요 데이터 항목

* 영업이익
* 순이익

데이터를 기반으로 기업의 **성장률 및 투자 점수**를 계산했습니다.

---

# ⚙️ 분석 방법

## 1️⃣ 성장률, 수익성 계산

기업의 성장성을 평가하기 위해 **3년 평균 성장률(CAGR)**을 계산했습니다.

대표적인 성장 지표

* 영업이익 성장률
* 순이익 성장률

수익성 지표

* 내후년 영업이익 기반 PER
* 내후년 순이익 기반 PER

---

## 2️⃣ 투자 점수 산출

각 성장 지표, 수익성 지표를 기반으로 기업별 투자 점수를 계산했습니다.

예시 방식

```
종합 점수 = 성장성 점수 + 수익성 점수
```

점수를 기준으로 종목을 랭킹화하여 포트폴리오를 구성했습니다.

---

## 3️⃣ 포트폴리오 구성

투자 점수가 높은 기업을 선별하여 포트폴리오를 구성했습니다.

예시 전략

* 상위 N개 종목 투자
* 점수 기반 가중 포트폴리오
